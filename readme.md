Recipe Hub - My Refactoring Journey

Welcome to Recipe Hub! This project is a special one for me because it shows how much I’ve improved in web development. I started this with just basic HTML, and honestly, it looked like something from the 90s. But then, I decided to dive deep into CSS to give it a modern, professional look.

What’s This Project About?
This is a simple yet clean recipe portal where I share my favorite dishes. The main goal was to practice how to take a "raw" HTML file and turn it into a modern UI using current design principles.

 What I Improved (The "Level Up" Part)
-From Messy to Clean Architecture:** I completely restructured the HTML. I replaced the old-school layout with a clean `div` based architecture, making the code much easier to read and maintain.
-Table Makeover: Before, my tables looked like boring Excel sheets. I removed the clunky borders and added a modern, minimalist feel with clean lines and a professional color palette.
-Flexbox Power:I used Flexbox to align icons, preparation times, and servings. Now everything is perfectly aligned and stays consistent across different sections—which was a huge headache at first!
-The "Card" Look & Depth: I wrapped each recipe in a white "card" with soft shadows (`box-shadow`). This creates a sense of depth and makes the content pop against the background.
-Color Palette & Branding: I moved away from default colors and implemented a unified theme using a professional deep burgundy (#7b284f). It gives the site a "branded" feel.
-Mobile Friendly:I added Media Queries so you can check these recipes on your phone while cooking without any layout issues.
-Semantic Polish: I fixed the header hierarchy (goodbye `h5`, hello `h2`!) and cleaned up unnecessary tags like `<hr>` to let the design breathe.
