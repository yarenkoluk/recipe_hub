 🥗 Recipe Hub 

This is a simple, structured recipe website that I built while learning the fundamentals of HTML.

🚀 About the Project
 It features healthy recipes like Broccoli Salad and Baked Potatoes, showcasing the core structure of a website before adding styles.

## 🛠️ Features & HTML Tags Used
In this project, I practiced and implemented:
* **Structural Tags:** `<div>`, `<nav>`, `<footer>` for layout.
* **Text Formatting:** `<h1>` to `<h3>`, `<strong>`, `<em>`, and `<blockquote>` for chef's notes.
* **Lists:** Ordered (`<ol>`) and Unordered (`<ul>`) lists for ingredients and steps.
* **Media:** Displaying images using `<img>`.
* **Data Representation:** Creating a nutritional value table with `<table>`, `<tr>`, `<th>`, and `<td>` (including `colspan` usage).
* **User Interaction:** A feedback form with `<input>`, `radio` buttons, and a `submit` button.
* **Links:** Navigation and external resource links using `<a>`.

## 📸 Preview
*Currently, this project is in its **HTML-only** phase. It represents the "skeleton" of a website. The next step is to beautify it using CSS!*
